
#### 启动
- 安装 **vue cli** 快速原型工具 
```
npm install -g @vue/cli-service-global
```
- 进入对应目录下，启动即可
```
vue serve ./animation.vue -o
```

#### 打包
```
vue build ./animation.vue
```

#### 使用
vue组件，拷贝至vue工程直接使用,也可自己改造成html/js
